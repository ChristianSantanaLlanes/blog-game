Usaremos **MSI Afterburner** para este tutorial. Es la herramienta estándar y funciona con cualquier marca.

### Pasos Básicos
1. Aumenta el "Power Limit" al máximo.
2. Sube el "Core Clock" de 50 en 50 Mhz.
3. Ejecuta un benchmark (como Heaven) después de cada subida.
4. Si ves artefactos visuales, baja un poco la frecuencia.

¡Recuerda monitorear siempre las temperaturas!
